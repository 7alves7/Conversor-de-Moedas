# 💱 Conversor de Moedas em Python

### 🚀 Sobre o Projeto
Este é um projeto desenvolvido durante meus estudos em **ADS**. Trata-se de um conversor de moedas funcional que utiliza Python para realizar cálculos de conversão de forma simples e eficiente.

O projeto foi desenvolvido e testado no **Google Colab**.

### 🛠️ Tecnologias e Conceitos Utilizados
- **Python**: Linguagem principal.
- **Lógica de Programação**: Estruturas de entrada, processamento e saída.
- **Google Colab**: Ambiente de desenvolvimento.
