# 💱 Conversor de Moedas em Python (com Groq API)

### 🚀 Sobre o Projeto
Este é um projeto desenvolvido durante meus estudos em **ADS**. Trata-se de um conversor de moedas funcional que utiliza Python e a tecnologia da **Groq Cloud** para realizar processamentos inteligentes e conversões de forma eficiente.

O projeto foi desenvolvido e testado no **Google Colab**.

### 🛠️ Tecnologias e Conceitos Utilizados
- **Python**: Linguagem principal.
- **Groq API**: Utilizada para processamento e lógica avançada.
- **Lógica de Programação**: Estruturas de entrada, processamento e saída.
- **Google Colab**: Ambiente de desenvolvimento.

---

### 🔑 Configuração Obrigatória (API Key)

Para que o código funcione, o utilizador precisa de uma chave de API do Groq. Siga os passos abaixo:

1.  Aceda ao site oficial: [Groq Cloud Console](https://console.groq.com/).
2.  Crie uma conta ou faça login.
3.  Vá até à secção **API Keys**.
4.  Clique em **"Create API Key"**, dê um nome à chave e copie o código gerado.
5.  No código Python (ou nos segredos do Colab), insira esta chave onde for solicitado para autenticar as requisições.
